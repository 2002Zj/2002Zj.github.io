1.寻找两数之和（已OK）

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

思路：a+b=target，使用一个hash表保存已经遍历的元素值以及对应下标，然后得到target和当前nums[i]的差值，再map集合中寻找对应的值，返回i和集合中对应值得坐标

class Solution {
    public int[] twoSum(int[] nums, int target) {
        //利用hash表，将遍历的元素放入hash表中，然后寻找target-num[i]的差值返回对应的坐标
        Map<Integer,Integer> map=new HashMap<>();
        int[] res=new int[2];//用于存放结果
        for(int i=0;i<nums.length;i++){
            int temp=target-nums[i];//得到差值
            if(map.containsKey(temp)){
                res[0]=i;
                res[1]=map.get(temp);
                break;//退出循环
            }else{
                map.put(nums[i],i);
            }
        }
        return res;
    }
}

2.字母异位词分组（还需复习）

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

示例 1:

输入: strs = [“eat”, “tea”, “tan”, “ate”, “nat”, “bat”]
输出: [[“bat”],[“nat”,”tan”],[“ate”,”eat”,”tea”]]

思路：字母异位词，字符排序后得到得字符串一定相同，利用这个特性，将互为字母异位词得单词分到一组

class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        Map<String,List<String>>map=new HashMap<>();
        for(String str:strs){
            char[] chars=str.toCharArray();
            Arrays.sort(chars);//排序
            String key=new String(chars);
            List<String>list=map.getOrDefault(key,new ArrayList<String>());
            list.add(str);
            map.put(key,list);
        }
        return new ArrayList<List<String>>(map.values());
    }
}

3.最长连续序列（还需复习）

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度

输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

思路：利用一个集合，先进行去重操作，遍历集合判断当前元素是否当前子序列的最小值，如果是则继续向后操作；

​ 然后判断当前元素的的值curNum的curNum+1是否存在，若存在则将当前序列的长度进行+1操作，直到无法找到连续子序列的下 一个值，然后进行长度比较。

class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for (int i = 0; i < nums.length; i++) { // 先去重
            set.add(nums[i]);
        }
        int lenMax = 0;
        for (int num : set) {
            if (!set.contains(num - 1)) { // 当前为子序列的最小值
                int curLen = 1;// 至少为1
                int curNum = num;
                while (set.contains(curNum + 1)) {
                    curNum++;
                    curLen++;
                }
                lenMax = Math.max(lenMax, curLen);
            }
        }
        return lenMax;

    }
}

4.移动零（已OK）

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]

思路：通过遍历元素，将非零元素全部交换到数组前部分，后半部分直接进行赋0操作；需要借助两个指针，left和right;

right：遍历数组；left：记录当前前半部分最后一个非零元素的索引下标

class Solution {
    public void moveZeroes(int[] nums) {
        //利用双指针，right遍历，left记录需要置零元素的起始位置的前一个位置
        int left=0,right=0;
        while(right<nums.length){
            if(nums[right]!=0){
                nums[left]=nums[right];
                left++;
            }
            right++;
        }
        while(left<nums.length){
            nums[left++]=0;
        }
    }
}

5.盛最多水的容器（已OK）

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

思路：一样是利用双指针，通过比较height[left] 和 height[right]的值来进行容量的计算，以及left和right的移动情况

class Solution {
    public int maxArea(int[] height) {
        int n=height.length;
        int left=0,right=n-1;
        int maxArea=0;
        while(left<right){
            if(height[left]<=height[right]) {
                maxArea=Math.max(maxArea,height[left]*(right-left));
                left++;
            }else{
                maxArea=Math.max(maxArea,height[right]*(right-left));
                right--;
            }
        }
        return maxArea;
    }
}

15.三数之和（还需复习）

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

思路：和两数之和有点相似，不过是多了一个数进行处理，需要用到两个指针，记住要进行去重处理

class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        // 一个for循环遍历数组，定义left和right,sum=nums[i]+nums[left]+nums[right]
        // 当sum<0;left+1;当sum>0时，right-1;sum=0时候，收获结果，记得去重
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        if (nums[0] > 0) return res;
        for (int i = 0; i < nums.length; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            int left = i + 1, right = nums.length - 1;
            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if (sum < 0) {
                    left++;
                } else if (sum > 0) {
                    right--;
                } else {
                    res.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    while (left < right && nums[left]==nums[left+1]) left++;
                    while (left < right && nums[right]==nums[right-1]) right--;
                    left++;
                    right--;
                }
            }
        }
        return res;
    }
}

42.接雨水（还需复习）

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6

解法1：利用双指针记录前缀最大高度和后缀最大高度（preMax和sufMax）表示，若preMax<sufMax，那么接水量的高度最高位preMax,再用preMax-当前的高度，得到当前位置的接水量，反之preMax>=sufMax也用同样方式处理，最后将接水量累加起来。

解法2：利用单调栈的特性，我们要找到当前位置的接水量，就要找到左边大于当前高度最近的元素和右边大于当前高度最近的元素，才能接住水。那么就可以利用单调栈来处理。具体解法如下：

class Solution {
    public int trap(int[] height) {
        // 解法1：双指针
        // int n = height.length;
        // int left = 0, right = n - 1;
        // int res = 0;// 储水量
        // int preMax = 0, sufMax = 0;// 前缀最大值和后缀最大值
        // while (left <= right) {
        // preMax = Math.max(preMax, height[left]);
        // sufMax = Math.max(sufMax, height[right]);
        // if (preMax < sufMax) {
        // res += preMax - height[left];
        // left++;
        // } else {
        // res += sufMax - height[right];
        // right--;
        // }
        // }
        // return res;
        // 解法2：利用单调栈，栈中维护一个递增的栈，栈顶为最小元素的索引
        //注意:栈中存放是索引，而不是值
        Deque<Integer> stack = new LinkedList<>();
        int n = height.length;
        int res = 0;
        for (int i = 0; i < n; i++) {
            while (!stack.isEmpty() && height[i] > height[stack.peek()]) {
                int top = stack.pop();// 取出栈顶元素
                if (stack.isEmpty()) {
                    break;
                }
                int left = stack.peek();// 左边大于height[i]最近的元素的索引
                int curWidth = i - left - 1;// 宽
                int curHeight = Math.min(height[left], height[i]) - height[top]; // 高
                res += curHeight * curWidth;
            }
            stack.push(i);
        }
        return res;
    }
}

347.前 K 个高频元素(还需复习)

给你一个整数数组 nums 和一个整数 k ，请你返回其中出现频率前 k 高的元素。你可以按 任意顺序 返回答案。

输入: nums = [1,1,1,2,2,3], k = 2
输出: [1,2]

思路：解决例题类型包括堆排序、求第 K 大的数、带权图的遍历等可以用到优先队列。思想就是维护了一个固定大小的堆。本题我们需要维护一个小根堆，每次当前元素的出现频率比小根堆的根节点大时候，弹出堆首元素，并将当前元素加入进去。

class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        //使用优先级队列PriorityQueue,因为优先级队列底层就是使用了堆排序，也可以自己实现小根堆排序
        Map<Integer,Integer>map=new HashMap<>();
        for (int num : nums) {
            map.put(num, map.getOrDefault(num, 0)+1);
        }
        PriorityQueue<int[]>queue=new PriorityQueue<>((pair1,pair2)->pair1[1]-pair2[1]);
        Set<Map.Entry<Integer, Integer>> entries = map.entrySet();
        for (Map.Entry<Integer, Integer> entry : entries) {
            if (queue.size()<k){
                queue.add(new int[]{entry.getKey(),entry.getValue()});
            }else {
                if (entry.getValue()>queue.peek()[1]){
                    queue.poll();
                    queue.add(new int[]{entry.getKey(),entry.getValue()});
                }
            }
        }
        int[] res=new int[k];
        for (int i = k-1; i>=0 ; i--) {
            res[i]=queue.poll()[0];
        }
        return res;
    }
}

74.搜索二维矩阵(还需复习)

给你一个满足下述两条属性的 m x n 整数矩阵：

  • 每行中的整数从左到右按非严格递增顺序排列。
  • 每行的第一个整数大于前一行的最后一个整数。

给你一个整数 target ，如果 target 在矩阵中，返回 true ；否则，返回 false 。

输入：matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3
输出：true

思路：暴力搜索+二分查找，通过对于每一行进行二分查找的方式。时间复杂度：O(mlogn)

class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        //暴力搜索+二分搜索
        for(int[] nums :matrix){
            if(serarch(nums,target)){
                return true;
            }
        }
        return false;
    }
    //二分搜索
    public boolean serarch(int[] nums,int target){
        int left=0,right=nums.length-1;
        while(left<=right){
            int mid=(right-left)/2+left;
            if(nums[mid]==target) return true;
            if(nums[mid]<target) left=mid+1;
            if(nums[mid]>target) right=mid-1;
        }
        return false;
    }
}

1052. 爱生气的书店老板

有一个书店老板，他的书店开了 n 分钟。每分钟都有一些顾客进入这家商店。给定一个长度为 n 的整数数组 customers ，其中 customers[i] 是在第 i 分钟开始时进入商店的顾客数量，所有这些顾客在第 i 分钟结束后离开。

在某些时候，书店老板会生气。 如果书店老板在第 i 分钟生气，那么 grumpy[i] = 1，否则 grumpy[i] = 0。

当书店老板生气时，那一分钟的顾客就会不满意，若老板不生气则顾客是满意的。

书店老板知道一个秘密技巧，能抑制自己的情绪，可以让自己连续 minutes 分钟不生气，但却只能使用一次。

请你返回 这一天营业下来，最多有多少客户能够感到满意 。

输入：customers = [1,0,1,2,1,1,7,5], grumpy = [0,1,0,1,0,1,0,1], minutes = 3
输出：16
解释：书店老板在最后 3 分钟保持冷静。
感到满意的最大客户数量 = 1 + 1 + 1 + 1 + 7 + 5 = 16.

解题思路：

通过题目分析，可以使用定长的滑动窗口来解决，先记录抑制情绪前的客户满意的数量，因为minutes是固定长度的，通过滑动固定窗口的来记录抑制情绪后所能够增加客户满意的最大数量，最后返回初始记录的数量和增加的最大的数量的和

代码：

class Solution {
    public int maxSatisfied(int[] customers, int[] grumpy, int minutes) {
        //滑动窗口先计算抑制情绪前满意的客户，然后再计算定长子数组中
        //客户不满意--->满意的增加的最大客户量，最后相加得到总的感到满意的客户数
        int total=0;
        for(int i=0;i<customers.length;i++){
            if(grumpy[i]==0){
                total+=customers[i];
            }
        }
        int increase=0;
        int maxIncrease=0;
        for(int i=0;i<customers.length;i++){
            increase+=customers[i]*grumpy[i];
            if(i>=minutes){ //维护滑动窗口的长度
                increase-=customers[i-minutes]*grumpy[i-minutes];
            }
            maxIncrease=Math.max(maxIncrease,increase);
        }
        return total+maxIncrease;
    }
}

70. 爬楼梯

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

输入：n = 2
输出：2
解释：有两种方法可以爬到楼顶。
1. 1 阶 + 1 阶
2. 2 阶

解题思路：

就是一个动态规划问题，并且如果改成每次爬1~n阶级就编程了一个完全背包问题。先确定dp数组的含义：dp[i]表示到第i阶有多少种方法。每次可以上1个或2个台阶，所以上到第i阶的方法数由i-1和i-2来决定，递推公式：dp[i]=dp[i-1]+dp[i-2]。初始化：dp[1]=1，dp[2]=2;

代码：

class Solution {
    public int climbStairs(int n) {
        //每次可以上1个或2个台阶，所以上到第i阶的方法数由i-1和i-2来决定
        //dp[i]表示上到第n阶有dp[i]种方法数
        //递推公式：dp[i]=dp[i-1]+dp[i-2];
        if(n<=2) return n;
        int[] dp=new int[n+1];
        //初始化dp数组
        dp[1]=1;
        dp[2]=2;
        for(int i=3;i<=n;i++){
            dp[i]=dp[i-1]+dp[i-2];
        }
        return dp[n];
    }
}

377. 组合总和IV

给你一个由 不同 整数组成的数组 nums ，和一个目标整数 target 。请你从 nums 中找出并返回总和为 target 的元素组合的个数。

输入：nums = [1,2,3], target = 4
输出：7
解释：
所有可能的组合为：
(1, 1, 1, 1)
(1, 1, 2)
(1, 2, 1)
(1, 3)
(2, 1, 1)
(2, 2)
(3, 1)
请注意，顺序不同的序列被视作不同的组合。

解题思路：

这是动态规划种的完全背包问题，target是背包，nums是物品物品可以重复使用，问装满背包的方法数有多少种。并且这题求得是排列数，对于物品和背包得遍历顺序有一定的要求。

确定dp数组含义：dp[i]表示装满容量为i的背包的方法数；递推公式：dp[i]+=dp[i-nums[j]]；初始化：dp[0]=1;

代码：

class Solution {
    public int combinationSum4(int[] nums, int target) {
        int[] dp=new int[target+1];
        dp[0]=1;
        for(int i=1;i<=target;i++){ //先遍历背包
            for(int j=0;j<nums.length;j++){ //再遍历物品
                if(nums[j]<=i){
                    dp[i]+=dp[i-nums[j]];
                }
            }
        }
        return dp[target];
    }
}

125.验证回文串

如果在将所有大写字符转换为小写字符、并移除所有非字母数字字符之后，短语正着读和反着读都一样。则可以认为该短语是一个 回文串 。字母和数字都属于字母数字字符。

给你一个字符串 s，如果它是 回文串 ，返回 true ；否则，返回 false 。

输入: s = "A man, a plan, a canal: Panama"
输出：true
解释："amanaplanacanalpanama" 是回文串。

解题思路：

对于Java中字符常用的API的掌握，定义两个指针left和right,左右同时遍历，当向右遍历时，若是非字母或数字，就left++,同理向左遍历时也一样操作，当left和right都指向字母或数字时候，比较两个字母的小写是否相同或数字是否相同，一旦遇到不同情况就返回false。

代码：

class Solution {
    public boolean isPalindrome(String s) {
        int left = 0, right = s.length() - 1;
        char[] chars = s.toCharArray();
        while (left < right) {
            //isLetterOrDigit()判断是都是字母或数字，是返回true，否则返回false
            while (left < right && !Character.isLetterOrDigit(chars[left])) {
                left++;
            }
            while (left < right && !Character.isLetterOrDigit(chars[right])) {
                right--;
            }
            //toLowerCase()将字母转为小写字母，若是数字则不变
            if (Character.toLowerCase(chars[left]) != Character.toLowerCase(chars[right])) {
                return false;
            }
            left++;
            right--;
        }
        return true;
    }
}

392. 判断子序列

给定字符串 s 和 t ，判断 s 是否为 t 的子序列。

字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，"ace"是"abcde"的一个子序列，而"aec"不是）。

示例 1：

输入：s = "abc", t = "ahbgdc"
输出：true

示例 2：

输入：s = "axc", t = "ahbgdc"
输出：false

解题思路：

借助一个辅助指针（index）指向s，遍历t字符串，若当前字符和s中index索引对应的字母相同，index++，直到若 index>=s.length()，说明存在子序列，放回true,否则返回false。

代码：

class Solution {
    public boolean isSubsequence(String s, String t) {
        if(s.length()==0) return true;
        int index=0;
        for(int i=0;i<t.length();i++){
            if(s.charAt(index)==t.charAt(i)){
                index++;
            }
            if(index>=s.length()){
                return true;
            }
        }
        return false;
    }
}

167. 两数之和 II - 输入有序数组

给你一个下标从 1 开始的整数数组 numbers ，该数组已按 非递减顺序排列 ，请你从数组中找出满足相加之和等于目标数 target 的两个数。如果设这两个数分别是 numbers[index1] 和 numbers[index2] ，则 1 <= index1 < index2 <= numbers.length 。

以长度为 2 的整数数组 [index1, index2] 的形式返回这两个整数的下标 index1 和 index2。

你可以假设每个输入 只对应唯一的答案 ，而且你 不可以 重复使用相同的元素。

你所设计的解决方案必须只使用常量级的额外空间。

输入：numbers = [2,7,11,15], target = 9
输出：[1,2]
解释：2 与 7 之和等于目标数 9 。因此 index1 = 1, index2 = 2 。返回 [1, 2] 。

解题思路：

利用双指针，一个left和一个right，左右向中间遍历，用一个sum记录当前 numbers[left]+numbers[right]的和，若sum>target，right向左靠近，若sum<target，left向右靠近，若sum==target，将left和right的坐标加1记录到一个大小为2的数组中。

代码：

class Solution {
    public int[] twoSum(int[] numbers, int target) {
        int left=0,right=numbers.length-1;
        int sum=0;
        int[] res=new int[2];
        while(left<right){
            sum=numbers[left]+numbers[right];
            if(sum>target){
                right--;
            }else if(sum<target){
                left++;
            }else{
                res[0]=left+1;
                res[1]=right+1;java
            }
        }
        return res;
    }
}

29. 两数相除

给你两个整数，被除数 dividend 和除数 divisor。将两数相除，要求 不使用 乘法、除法和取余运算。

整数除法应该向零截断，也就是截去（truncate）其小数部分。例如，8.345 将被截断为 8 ，-2.7335 将被截断至 -2 。

返回被除数 dividend 除以除数 divisor 得到的 商 。

注意：假设我们的环境只能存储 32 位 有符号整数，其数值范围是 [−231, 231 − 1] 。本题中，如果商 严格大于 231 − 1 ，则返回 231 − 1 ；如果商 严格小于 -231 ，则返回 -231 。

输入: dividend = 10, divisor = 3
输出: 3
解释: 10/3 = 3.33333.. ，向零截断后得到 3 。

解题思路：

本题禁止使用除法、乘法、取余运算。那么只能使用除法的本质-减法运算，由于时间限制，通过倍增的方式来加快运算速度

class Solution {
    public int divide(int dividend, int divisor) {

        boolean isNegative = (dividend > 0 ^ divisor > 0);
        if (dividend == Integer.MIN_VALUE && divisor == -1) return Integer.MAX_VALUE;

        long dvd = Math.abs((long) dividend);
        long dvs = Math.abs((long) divisor);
        int quotient = 0;

        while (dvd >= dvs) {
            long temp = dvs, multiplier = 1;

            // 通过翻倍除数并检查是否可以继续减去来加速计算过程
            //相当于翻倍除数，得到除后得到余数重新从divisor继续除
            while (dvd >= (temp << 1)) {
                temp <<= 1;
                multiplier <<= 1;
            }

            dvd -= temp;
            quotient += multiplier;
        }

        return isNegative ? -quotient : quotient;
    }
}

27. 移除元素

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2]
解释：函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。你不需要考虑数组中超出新长度后面的元素。例如，函数返回的新长度为 2 ，而 nums = [2,2,3,3] 或 nums = [2,2,0,0]，也会被视作正确答案。

解题思路：

解题思路：

利用双指针，定义一个slow指针，遍历数组，若当前元素的值不等于val值，则nums[slow]=nums[i]，并且slow++，最后返回slow，就是一处val元素后的数组大小。

代码：

class Solution {
    public int removeElement(int[] nums, int val) {
        //利用双指针，一个指针遍历数组，一个用来不等于val时，需要交换的标记
        int slow=0;
        for(int i=0;i<nums.length;i++){
            if(nums[i]!=val){
                nums[slow++]=nums[i];
            }
        }
        return slow;
    }
}
