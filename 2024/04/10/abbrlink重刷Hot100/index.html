1.寻找两数之和（已OK）

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

思路：a+b=target，使用一个hash表保存已经遍历的元素值以及对应下标，然后得到target和当前nums[i]的差值，再map集合中寻找对应的值，返回i和集合中对应值得坐标

class Solution {
    public int[] twoSum(int[] nums, int target) {
        //利用hash表，将遍历的元素放入hash表中，然后寻找target-num[i]的差值返回对应的坐标
        Map<Integer,Integer> map=new HashMap<>();
        int[] res=new int[2];//用于存放结果
        for(int i=0;i<nums.length;i++){
            int temp=target-nums[i];//得到差值
            if(map.containsKey(temp)){
                res[0]=i;
                res[1]=map.get(temp);
                break;//退出循环
            }else{
                map.put(nums[i],i);
            }
        }
        return res;
    }
}

2.字母异位词分组（还需复习）

给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

字母异位词 是由重新排列源单词的所有字母得到的一个新单词。

示例 1:

输入: strs = [“eat”, “tea”, “tan”, “ate”, “nat”, “bat”]
输出: [[“bat”],[“nat”,”tan”],[“ate”,”eat”,”tea”]]

思路：字母异位词，字符排序后得到得字符串一定相同，利用这个特性，将互为字母异位词得单词分到一组

class Solution {
    public List<List<String>> groupAnagrams(String[] strs) {
        Map<String,List<String>>map=new HashMap<>();
        for(String str:strs){
            char[] chars=str.toCharArray();
            Arrays.sort(chars);//排序
            String key=new String(chars);
            List<String>list=map.getOrDefault(key,new ArrayList<String>());
            list.add(str);
            map.put(key,list);
        }
        return new ArrayList<List<String>>(map.values());
    }
}

3.最长连续序列（还需复习）

给定一个未排序的整数数组 nums ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度

输入：nums = [100,4,200,1,3,2]
输出：4
解释：最长数字连续序列是 [1, 2, 3, 4]。它的长度为 4。

思路：利用一个集合，先进行去重操作，遍历集合判断当前元素是否当前子序列的最小值，如果是则继续向后操作；

​ 然后判断当前元素的的值curNum的curNum+1是否存在，若存在则将当前序列的长度进行+1操作，直到无法找到连续子序列的下 一个值，然后进行长度比较。

class Solution {
    public int longestConsecutive(int[] nums) {
        Set<Integer> set = new HashSet<>();
        for (int i = 0; i < nums.length; i++) { // 先去重
            set.add(nums[i]);
        }
        int lenMax = 0;
        for (int num : set) {
            if (!set.contains(num - 1)) { // 当前为子序列的最小值
                int curLen = 1;// 至少为1
                int curNum = num;
                while (set.contains(curNum + 1)) {
                    curNum++;
                    curLen++;
                }
                lenMax = Math.max(lenMax, curLen);
            }
        }
        return lenMax;

    }
}

4.移动零（已OK）

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]

思路：通过遍历元素，将非零元素全部交换到数组前部分，后半部分直接进行赋0操作；需要借助两个指针，left和right;

right：遍历数组；left：记录当前前半部分最后一个非零元素的索引下标

class Solution {
    public void moveZeroes(int[] nums) {
        //利用双指针，right遍历，left记录需要置零元素的起始位置的前一个位置
        int left=0,right=0;
        while(right<nums.length){
            if(nums[right]!=0){
                nums[left]=nums[right];
                left++;
            }
            right++;
        }
        while(left<nums.length){
            nums[left++]=0;
        }
    }
}

5.盛最多水的容器（已OK）

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

思路：一样是利用双指针，通过比较height[left] 和 height[right]的值来进行容量的计算，以及left和right的移动情况

class Solution {
    public int maxArea(int[] height) {
        int n=height.length;
        int left=0,right=n-1;
        int maxArea=0;
        while(left<right){
            if(height[left]<=height[right]) {
                maxArea=Math.max(maxArea,height[left]*(right-left));
                left++;
            }else{
                maxArea=Math.max(maxArea,height[right]*(right-left));
                right--;
            }
        }
        return maxArea;
    }
}

6.三数之和（还需复习）

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。请你返回所有和为 0 且不重复的三元组。

思路：和两数之和有点相似，不过是多了一个数进行处理，需要用到两个指针，记住要进行去重处理

class Solution {
    public List<List<Integer>> threeSum(int[] nums) {
        // 一个for循环遍历数组，定义left和right,sum=nums[i]+nums[left]+nums[right]
        // 当sum<0;left+1;当sum>0时，right-1;sum=0时候，收获结果，记得去重
        List<List<Integer>> res = new ArrayList<>();
        Arrays.sort(nums);
        if (nums[0] > 0) return res;
        for (int i = 0; i < nums.length; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }
            int left = i + 1, right = nums.length - 1;
            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];
                if (sum < 0) {
                    left++;
                } else if (sum > 0) {
                    right--;
                } else {
                    res.add(Arrays.asList(nums[i], nums[left], nums[right]));
                    while (left < right && nums[left]==nums[left+1]) left++;
                    while (left < right && nums[right]==nums[right-1]) right--;
                    left++;
                    right--;
                }
            }
        }
        return res;
    }
}

7.接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6

解法1：利用双指针记录前缀最大高度和后缀最大高度（preMax和sufMax）表示，若preMax<sufMax，那么接水量的高度最高位preMax,再用preMax-当前的高度，得到当前位置的接水量，反之preMax>=sufMax也用同样方式处理，最后将接水量累加起来。

解法2：利用单调栈的特性，我们要找到当前位置的接水量，就要找到左边大于当前高度最近的元素和右边大于当前高度最近的元素，才能接住水。那么就可以利用单调栈来处理。具体解法如下：

class Solution {
    public int trap(int[] height) {
        // 解法1：双指针
        // int n = height.length;
        // int left = 0, right = n - 1;
        // int res = 0;// 储水量
        // int preMax = 0, sufMax = 0;// 前缀最大值和后缀最大值
        // while (left <= right) {
        // preMax = Math.max(preMax, height[left]);
        // sufMax = Math.max(sufMax, height[right]);
        // if (preMax < sufMax) {
        // res += preMax - height[left];
        // left++;
        // } else {
        // res += sufMax - height[right];
        // right--;
        // }
        // }
        // return res;
        // 解法2：利用单调栈，栈中维护一个递增的栈，栈顶为最小元素的索引
        //注意:栈中存放是索引，而不是值
        Deque<Integer> stack = new LinkedList<>();
        int n = height.length;
        int res = 0;
        for (int i = 0; i < n; i++) {
            while (!stack.isEmpty() && height[i] > height[stack.peek()]) {
                int top = stack.pop();// 取出栈顶元素
                if (stack.isEmpty()) {
                    break;
                }
                int left = stack.peek();// 左边大于height[i]最近的元素的索引
                int curWidth = i - left - 1;// 宽
                int curHeight = Math.min(height[left], height[i]) - height[top]; // 高
                res += curHeight * curWidth;
            }
            stack.push(i);
        }
        return res;
    }
}
